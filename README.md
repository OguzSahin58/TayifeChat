# Global Chatting

## Overview
A personal chat application to communicate with friends while learning C++ with help from Claude AI.

## Project Goals
- Build a personal chat application with friends
- Learn C++ coding with AI assistance from Claude
- Understand network programming and socket communication

## How to Run

### Compile
```bash
g++ server.cpp -o server.exe -lws2_32
g++ client.cpp -o client.exe -lws2_32
```

### Execute
```bash
./server.exe
./client.exe
```
